MapIO

This function maps a region of physical memory into the linear address space of the application. 

PVOID _stdcall MapIO(      
   PVOID winio_handle,     
   ULONG physical_address, 
   ULONG size              
);

Parameters

winio_handle
[in] Pointer retourned by InitializeWinIo 

physical_address
[in] Pointer to the physical address

size
[in] Number of bytes to map

Return Values

If the function succeeds, the return value is the linear address corresponding to the physical address that was passed in pbPhysAddr. Otherwise, the return value is NULL.  The result should be used when calling UnMapIO

Remarks

The MapIO function maps a region of physical memory into the linear address space of the calling application.

Note: Currently not implemented on Windows 98/ME.
